# Online-Food-Ordering-Management-System-using-SQL
Online Food Ordering Management System using SQL focuses mainly on managing online food orders. To be more precise, this system helps to keep track of customers, and their orders.
Created a SQL database for an Online Food Ordering Management System to manage customer, restaurant, cuisine, admin, menu items and orders data using SQL queries.
Performed Analysis on Online Food Ordering Data and obtained insights from it by using various SQL queries, Joins, Window Functions, Views and Stored Procedures.
